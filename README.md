首次
1.安装taobao的 cnpm
npm install -g cnpm --registry=https://registry.npm.taobao.org

2.安装vue-cli
cnpm install -g vue-cli

3.创建项目

vue init webpack + 项目名称

4.下载依赖
cd到项目名称下
cnpm install

5.启动项目
npm run dev
